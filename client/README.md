# Frontend (Client)

Aplicação frontend em Vue 3 + Vite.

## Pré-requisitos

- Node.js instalado (compatível com `^20.19.0` ou `>=22.12.0`)
- pnpm instalado

Instalar o pnpm (se necessário):

```bash
npm install -g pnpm
```

## Instalação

Execute dentro da pasta `client/`:

```bash
cd client
pnpm install
```

## Rodar em desenvolvimento

Dentro de `client/`:

```bash
pnpm dev
```

## Build de produção

Dentro de `client/`:

```bash
pnpm build
```

## Preview local da build

Dentro de `client/`:

```bash
pnpm preview
```

## Fluxo rápido (resumo)

```bash
cd client
pnpm install
pnpm dev
```
